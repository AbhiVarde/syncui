### October 2024 - First launch

I am thrilled to introduce Sync UI, a collection of beautifully designed UI components that help you build business websites faster and easier. Built with MUI and Framer Motion, sync ui offers seamless integration and stunning animations. Our library is accessible, customizable, free, and open-source.

#### Initial components include:

- [Avatars](/docs/avatars)
- [Background animations](/docs/backgrounds)
- [Loaders](/docs/loaders)
- [Separators](/docs/separators)
- [Buttons](/docs/buttons)
- [Text animations](/docs/texts)

#### Key features:

- Beautifully designed components
- Built with MUI and Framer Motion
- Accessible and customizable
- Free and open-source
- Ready for your next project

You can copy and paste the code and customize it according to your needs. Feel free to give feedback and contribute to the project at [GitHub](https://github.com/AbhiVarde/syncui)

Sync UI continues to evolve with your feedback. Stay tuned for more exciting updates!
