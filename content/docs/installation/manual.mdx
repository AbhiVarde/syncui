#### Prerequisites

You'll need an existing React project with a package manager installed (npm, yarn, pnpm, or bun).

#### Install dependencies

Add the required packages for Sync UI to your project:

<PackageManagerTabs
  npm="npm install @mui/material @emotion/react @emotion/styled motion react-icons"
  yarn="yarn add @mui/material @emotion/react @emotion/styled motion react-icons"
  pnpm="pnpm add @mui/material @emotion/react @emotion/styled motion react-icons"
  bun="bun add @mui/material @emotion/react @emotion/styled motion react-icons"
/>

#### Use the CLI

Add a component or block directly into your project:

<PackageManagerTabs
  npm="npx @abhivarde/syncui@latest add accordion"
  yarn="yarn dlx @abhivarde/syncui@latest add accordion"
  pnpm="pnpm dlx @abhivarde/syncui@latest add accordion"
  bun="bunx --bun @abhivarde/syncui@latest add accordion"
/>

Add a specific variant with `name/variant`, for example `accordion/brutalist`.

#### Start development

Run your development server, the command depends on your project setup:

<PackageManagerTabs
  npm="npm start"
  yarn="yarn start"
  pnpm="pnpm start"
  bun="bun start"
/>

Or if using Vite or a custom setup:

<PackageManagerTabs
  npm="npm run dev"
  yarn="yarn dev"
  pnpm="pnpm dev"
  bun="bun dev"
/>
