#### Create a new project

Create a new Vite application with React:

<PackageManagerTabs
  npm="npm create vite@latest my-app -- --template react"
  yarn="yarn create vite my-app --template react"
  pnpm="pnpm create vite my-app --template react"
  bun="bun create vite my-app --template react"
/>

Navigate to your project:

<PackageManagerTabs
  npm="cd my-app"
  yarn="cd my-app"
  pnpm="cd my-app"
  bun="cd my-app"
/>

#### Install dependencies

First, install the base dependencies:

<PackageManagerTabs
  npm="npm install"
  yarn="yarn"
  pnpm="pnpm install"
  bun="bun install"
/>

Then add the required packages for Sync UI:

<PackageManagerTabs
  npm="npm install @mui/material @emotion/react @emotion/styled motion react-icons"
  yarn="yarn add @mui/material @emotion/react @emotion/styled motion react-icons"
  pnpm="pnpm add @mui/material @emotion/react @emotion/styled motion react-icons"
  bun="bun add @mui/material @emotion/react @emotion/styled motion react-icons"
/>

#### Start development

Launch your development server:

<PackageManagerTabs
  npm="npm run dev"
  yarn="yarn dev"
  pnpm="pnpm dev"
  bun="bun dev"
/>

Open [http://localhost:5173](http://localhost:5173) in your browser to see your app running.
