---

#### Create a new project

Create a new Next.js application:

<PackageManagerTabs
  npm="npx create-next-app@latest my-app"
  yarn="yarn create next-app my-app"
  pnpm="pnpm create next-app my-app"
  bun="bun create next-app my-app"
/>

Navigate to your project:

<PackageManagerTabs
  npm="cd my-app"
  yarn="cd my-app"
  pnpm="cd my-app"
  bun="cd my-app"
/>

#### Install dependencies

Add the required packages for Sync UI:

<PackageManagerTabs
  npm="npm install @mui/material @emotion/react @emotion/styled motion react-icons"
  yarn="yarn add @mui/material @emotion/react @emotion/styled motion react-icons"
  pnpm="pnpm add @mui/material @emotion/react @emotion/styled motion react-icons"
  bun="bun add @mui/material @emotion/react @emotion/styled motion react-icons"
/>

#### Start development

Launch your development server:

<PackageManagerTabs
  npm="npm run dev"
  yarn="yarn dev"
  pnpm="pnpm dev"
  bun="bun dev"
/>

Open [http://localhost:3000](http://localhost:3000) in your browser to see your app running.

#### You're all set!
