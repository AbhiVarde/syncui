#### Introduction

Sync UI is a collection of animated components and affordable templates that you can copy directly into your React projects. No installation required-just copy, paste, and customize.

#### Prerequisites

You'll need a React or Next.js project with these dependencies:

• **MUI** – A flexible and accessible UI component library  
• **Framer Motion** – Powerful animations for React  
• **React Icons** – Popular icon pack for React

#### Quick Setup

#### 1. Create Your Project

**React Project:**

```bash
npx create-react-app my-app
cd my-app
```

**Next.js Project:**

```bash
npx create-next-app@latest my-app
cd my-app
```

#### 2. Install Dependencies

```bash
npm install @mui/material @emotion/react @emotion/styled framer-motion react-icons
```

#### 3. Start Building

```bash
# React
npm start

# Next.js
npm run dev
```

#### How It Works

**Components:** Browse our component library, copy the code, and paste it directly into your project. Each component is ready to use and fully customizable.

**Templates:** Download complete template zip files with all components included. Modify and adapt them to match your design needs.

**No Lock-in:** All code is yours to modify. Change colors, animations, layouts-everything is customizable.
